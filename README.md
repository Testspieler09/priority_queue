This project contains the implementation of priority queues in C and Rust.

Each library implements the priority queue in two different versions.
- Min Heap Priority Queue
- Fibonacci Heap Priority Queue

> [!WARNING]
>
> The implementation contains some issues that I will not fix.
> It should not be seen as a template for implementing a priority queue but rather as an inspiration **at most**.
